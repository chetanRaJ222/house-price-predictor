# House Price Predictor

A machine learning project that predicts house prices using real-world housing data.

## Overview
This project uses a regression-based machine learning model trained on housing market data to estimate house prices based on numerical features such as population, income, and housing attributes.

## Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Google Colab  

## Features
- Data cleaning and preprocessing  
- Feature selection  
- Linear Regression model training  
- Model evaluation using R² score  
- Trained model saved for deployment  

## Files
- `House_Price_Predictor.ipynb` – Complete ML pipeline  
- `house_price_model.pkl` – Trained ML model  

## Result
The model achieves over **75% R² accuracy** on unseen test data, indicating strong predictive performance.
